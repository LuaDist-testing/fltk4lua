### `Valuator` (abstract):

Corresponds to:
[`Fl_Valuator`](http://www.fltk.org/doc-1.3/classFl__Valuator.html)

*   `obj:bounds( d, d )`
*   `obj:clamp( d ) ==> d`
*   `obj:format() ==> s`
*   `obj:increment( d, i ) ==> d`
*   `obj:range( d, d )`
*   `obj:round( d ) ==> d`
*   `obj.step  [get: d; set: d]`
*   `obj.value  [get: d; set: d]`
*   `obj.maximum  [get: d; set: d]`
*   `obj.minimum  [get: d; set: d]`
*   `obj.precision  [get: -; set: i]`
